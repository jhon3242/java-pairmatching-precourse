# 기능 구현 목록

## 페어 매칭 기능
- [ ] 페어 매칭은 두명씩 한다.
  - [ ] 페어 매칭 대상이 홀수인 경우 3인으로 구성
  - [ ] 같은 레벨에서 이미 페어를 맺은 크루랑은 다시 매칭 X
    - [ ] 이미 페어를 맺은 경우이면 크루 순서를 다시 셔플하여 앞에서부터 추출.
    - [ ] 3회 시도까지 매칭되지 않거나 할 경우의 수가 없는 경우 에러 출력

## 기능 선택
- [X] 기능 선택을 입력 받음
  - [X] `예외 사항` : 선택지가 없는 값을 입력한 경우
  
## 페어 매칭
- [X] 미션 등의 내용 출력
- [X] "과정, 레벨, 과제" 입력받음
- [X] 이미 매칭한 정보가 있으면 메시지 출력

## 페어 조회
- [ ] 과정 및 미션 내용들 출력
- [ ] "과정, 레벨, 과제" 입력받음
- [ ] 해당 매칭했던 결과 출력

## 페어 초기화
- [ ] "초기화 되었습니다." 출력
- [ ] 기존 매칭 결과 초기화
